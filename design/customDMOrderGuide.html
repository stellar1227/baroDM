맞춤DM 주문방법

타겟설정부터 결제까지 맞춤홍보 DM 주문 방법을 확인하세요.

  1. STEP 01 타겟 설정하기
    타겟 설정 기준을 선택합니다.
    • 선택하면 타겟 설정 레이어 창 오픈
    타겟 설정 레이어창
    • DM을 발송 할 지역을 선택합니다.
    타겟 설정 레이어창
    • 상세 조건을 설정합니다.
    발송 타겟 수량을 조정합니다.
    • 선택한 동과 상세 조건에 부합하는 가구의 수를 원하는 건 수만큼 조정합니다.
    • 수량 조정 후 ‘주소추가’를 클릭하면 발송 대상 목록이 생성됩니다.
    • 주소추가 후 다른 동을 선택하면 여러 동의 발송 타겟을 선택할 수 있습니다.
  2. STEP 02 디자인 선택 / 제작물 기본 정보 입력
    디자인 템플릿 선택 또는 디자인 파일 업로드
    • 원하는 디자인탬플릿을 선택하거나 가지고 있는 디자인 파일을 업로드합니다.
    • 원하는 방식의 탭을 선택하여 진행하시면 됩니다.

    디자인 파일 및 전단지 내용 파일 업로드 시 특이사항 메모

  3. STEP 03 발신자 정보 / 봉투 표기 내용 입력
    발신자 정보와 봉투 기재 정보 입력
    • 주문자 정보와 발신자 정보를 정확하게 입력해주세요.
    • 봉투 기재 내용은 DM 수신자가 보는 내용이므로 신중하게 입력해주세요.
  4. STEP 04 저장 및 결제
    주문
    • 결제를 바로 진행하시고 주문을 완료하시면 됩니다.
    주문 내역 저장
    • ‘My DM 보관함 > 신청 중인 주문’에 일주일 동안 보관됩니다.
    • 일주일 내에 결제를 진행하시면 주문이 완료됩니다.
    • 여러 건의 주문을 결제하실 때 이용하시면 편리합니다.
